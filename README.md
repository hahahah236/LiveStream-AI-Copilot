# LiveStream-AI-Copilot
本项目是一个基于逻辑分流的直播自动化复盘工具。核心解决直播日志数据信噪比低、复盘建议同质化、以及非业务输入干扰导致模型幻觉的问题。

核心工程逻辑
1. 动态逻辑路由 (Decision Logic)
系统通过工作流节点实时获取 is_hot 变量（热度判定），实现毫秒级策略分流：
Hot-Sale Mode (True)：切换至高频逼单策略，输出成交导向的复盘指令与短视频脚本。
Nurture Mode (False)：切换至信任构建策略，侧重品类教育与互动钩子，生成长留存建议。
2. 语义围栏 (Railguards)
在推理节点前置防御性逻辑，实现输入合法性校验：
异常拦截：自动识别乱码、跨领域话题（如天气、无关学术内容）并触发错误拦截，防止浪费 Token 或产生幻觉。
合规过滤：针对电商场景自动对极限词（如“最/第一”）进行非敏感化转换。
资产说明
/workflow：Coze 导出工程包。支持全节点逻辑导入。
/prompts：自适应提示词源码。包含防御性指令集与差异化分流逻辑。
/data：极端环境测试集。包含 hot_rich（高并发高干扰）与 cold_rich（低频次信任缺失）样本。

测试与复现
导入：将 /workflow 导入 Coze 平台。
场景验证：
输入 sample_hot_rich.txt -> 验证是否输出【收割模式】建议。
输入 sample_cold_rich.txt -> 验证是否输出【沉水模式】建议。
